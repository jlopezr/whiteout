# WhiteOut

WhiteOut is a simple macOS app designed to help clean your screen.

It displays a fullscreen background and lets you cycle through different modes so dust, fingerprints, streaks, and smudges are easier to see while wiping the display.

## Features

- Fullscreen overlay
- Multiple display modes:
  - White
  - Gray with dark scan line
  - Black with light scan line
- Moving vertical line to help reveal marks
- Keyboard controls:
  - `Space` to switch mode
  - `Esc` to quit
  - `Cmd+Q` to quit

## Use case

WhiteOut is useful when cleaning a monitor or laptop screen with a microfiber cloth and you want to quickly spot remaining marks under different contrast conditions.

## Build

This project can be built from the command line on macOS using Apple Command Line Tools:

```bash
make
```

Run it with:

```bash
make run
```
